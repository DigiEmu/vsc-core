# VSC v2.25 — Partner / Reviewer Interop Brief

Release: v2.25  
Project: DigiEmu VSC / VSC Core

## Summary

VSC v2.25 adds a compact partner-facing and reviewer-facing interoperability brief.

The brief provides a short entry point into the VSC interop model after the v2.17 through v2.24 interop arc.

## Added

New interop brief:

```text
docs/interop/PARTNER_REVIEWER_INTEROP_BRIEF.md
```

## Purpose

The brief helps external partners and reviewers understand:

- what VSC is
- what the current interop status is
- how TBN / VSC / CLARIXO boundaries remain separated
- what external references mean
- what external references do not mean
- how safe handoff language should be used
- what questions partners should answer before integration work
