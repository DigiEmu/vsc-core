# VSC v2.11 — CI Badge / README Conformance Status

Release: v2.11  
Project: DigiEmu VSC / VSC Core

## Summary

VSC v2.11 adds a GitHub Actions status badge to the README.

The badge exposes the current status of the `VSC Conformance` workflow directly from the repository front page.

## Added

README badge:

```markdown
[![VSC Conformance](https://github.com/DigiEmu/vsc-core/actions/workflows/vsc-conformance.yml/badge.svg)](https://github.com/DigiEmu/vsc-core/actions/workflows/vsc-conformance.yml)
```

## Release meaning

v2.11 makes the automated conformance status visible without opening the Actions tab first.

This completes the small CI visibility chain:

```text
v2.9   → CI Conformance Check
v2.10  → CI Artifact Upload
v2.11  → README CI Status Badge
```

## Unchanged

v2.11 does not change:

- Go verifier semantics
- Node runner comparison semantics
- GitHub Actions workflow logic
- Conformance fixture expected results
- PASS / FAIL / ERROR meaning
- Evidence bundle verification logic
