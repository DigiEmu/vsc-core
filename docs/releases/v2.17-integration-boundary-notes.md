## Summary

VSC v2.17 adds integration boundary notes for the TBN / VSC / CLARIXO three-layer model.

The new document explains how the layers can reference each other without redefining each other's proof boundaries.

## Important status note

This release does not claim that TBN, VSC, and CLARIXO are fully integrated.

It documents safe interoperability boundaries and reference patterns.

## Three-layer model

```text
TBN     = identity / trust verification
VSC     = decision-state verification
CLARIXO = responsibility / attribution context
```

## Core boundary principle

```text
Each layer may reference the others,
but no layer should redefine the proof boundary of another layer.
```

## VSC boundary

```text
VSC proves state integrity, not truth.
```

VSC does not prove identity, legal responsibility, attribution, fairness, moral correctness, or full compliance by itself.
