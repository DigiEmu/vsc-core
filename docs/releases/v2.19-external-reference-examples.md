# VSC v2.19 — External Reference Examples

Release: v2.19  
Project: DigiEmu VSC / VSC Core

## Summary

VSC v2.19 adds concrete example JSON files for the v2.18 external reference profile draft.

These examples show how VSC-related metadata can point to external artifacts without importing their proof claims.

## Added

New example directory:

```text
examples/external-references/
```

Example files:

```text
tbn-receipt-reference.json
clarixo-record-reference.json
external-audit-note-reference.json
partner-evidence-record-reference.json
other-reference.json
README.md
```

## Profile used

All examples use:

```text
vsc-external-reference-v2.18-draft
```

## Required proof boundary

All examples keep:

```json
{
  "proof_boundary": "external_reference_only",
  "verified_by_vsc": false
}
```

## Core boundary

```text
External references are pointers, not proof imports.
```
